# lab1_backend
# Платформа вибору вибіркових дисциплін
**Лабораторна робота №1 (Частина 1)**

### Опис проєкту
Адаптивний вебзастосунок для перегляду та вибору навчальних курсів. 
Реалізовано з використанням семантичної верстки та CSS Grid.

### Технології
* HTML5 (Семантичні теги)
* CSS3 (Grid, Flexbox, Media Queries)
* JavaScript (Burger-menu, animations)

### Як запустити
1. Склонуйте репозиторій: `git clone [посилання]`
2. Відкрийте `index.html` у браузері або запустіть через Node.js: `node server.js`.
